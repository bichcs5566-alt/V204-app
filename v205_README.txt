v205 固定網址 App 版 使用說明

這版重點：
1. 不手動選檔
2. 從固定 Base URL 自動抓取：
   - v202_nav.csv
   - v202_positions.csv
   - v202_summary.csv
3. 可安裝到手機主畫面
4. 有更新提示
5. 可要求瀏覽器通知（是否支援，依手機與瀏覽器而定）

一、你要放上 GitHub Pages 的檔案
- v205_fixed_url_app.html
- manifest.json
- service-worker.js
- v202_nav.csv
- v202_positions.csv
- v202_summary.csv

二、第一次用
1. 用 Safari 或 Chrome 打開 v205_fixed_url_app.html 的網址
2. 在頁面上輸入 Base URL
   例：https://你的帳號.github.io/V204-app
3. 按「儲存網址」
4. 按「立即更新」

三、變成桌面 App
iPhone：
- Safari → 分享 → 加入主畫面
Android：
- Chrome → 安裝應用程式 / 加入主畫面
